# AGENT_PRESET 默认配置（全局回退，人话版）

用途：当 `input/<目录>/` 下没有自定义 `AGENT_PRESET.md` 时，自动使用本文件。

这是默认直跑策略，尽量少配置、少打断：
1. 受众默认按“泛知识读者”处理
2. 故事基调默认“专业理性”
3. 视觉风格默认“活泼版”（仅当用户明确指定时才切换到专业版）
4. 生成人设默认自动路由（医学优先 医学知识编辑，非医学优先 通用知识编辑）
5. 生成范围默认“自动判定（series/single）”
6. 作者署名默认跟随当前人设（封面+结尾完整署名，内页默认不展示）
7. 标题语气默认“活泼网感”（医学主题保持术语准确，不做夸张承诺）

只有以下情况会先询问用户再继续：
1. 人设判断置信度低
2. 输入目录或系列名称不明确
3. 有明显术语冲突、口径冲突或硬约束缺失

可选覆盖（按需，不强制）：
- 目标受众
- 故事基调
- 风格偏好（专业版/活泼版/混合）
- 叙事风格指南（指定文件名或自动匹配）
- 故事指南（指定文件名或自动匹配）
- 指定人设（如固定使用 医学知识编辑）
- 指定只生成某几篇
- 禁用词、必须术语、品牌口径
