# 通用手册/指南风格速查卡（Quick）

对应长文：`通用指南类内容创作风格指南.md`

## 适用场景

- 规则解读、操作手册、流程指南、制度说明

## 风格定位

- 语气：清晰、严谨、可执行
- 目标：让读者从“看不懂”到“能操作”

## 推荐结构

1. 开头：本节目标与适用范围
2. 中段：规则 -> 原理 -> 示例 -> 特例/避坑
3. 结尾：要点清单 + FAQ/交叉引用

## 必做

1. 先总后分，层级稳定
2. 每条规则至少配一个示例
3. 明确适用边界与例外条件
4. 结尾提供速查总结

## 禁做

1. 模糊词（大概、可能、差不多）
2. 跳步解释，默认读者“已懂”
3. 只讲是什么，不讲为什么
