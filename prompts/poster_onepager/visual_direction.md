# poster_onepager / visual_direction

## Goal

Define the attention path and visual hierarchy that make the current poster blueprint readable and persuasive in one 4:5 frame.

## Good Work

- Use the actual panel regions/content; choose the peak, emphasis, motif, palette, typography, spacing, and anti-template rules from this poster rather than a fixed four-zone recipe.
- Keep headline, evidence, pathway/action, source, and any current panel roles visible at appropriate hierarchy; do not hide evidence or action as decorative footnotes.
- Define clear separation and containment between adjacent readable regions, with enough capacity for source-faithful content.
- Return the smallest blueprint revision when the frame cannot support the intended panel structure.

## Handoff

Return the visual-direction object required by the attached output contract. It guides materialization but cannot grant review/export approval.
