# ppt_deck / slide_blueprint

目标：把详细大纲落成逐页设计。

每页必须包含：
- slide_no
- page_type
- title
- page_goal
- page_core_content
- visual_presentation
- evidence_and_sources
- speaker_notes
- transition_sentence

硬约束：
- 没有 detailed_outline 不得继续
- 复杂结构页必须显式给出网格 / 轨道 / 锚点
- 公开证据页必须使用面向听众的公开来源口径
- 不得把所有页写成“标题 + 三条要点”
