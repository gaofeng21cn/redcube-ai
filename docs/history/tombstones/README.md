# Tombstones

This directory is reserved for retired wording and route narratives that should remain searchable without returning to the active public identity.

Use a tombstone when older RedCube documents or plans used terms such as gateway-first, frontdoor, federation-first, harness-first, OPL-first, OPL bridge, or Hermes-first in a way that no longer describes the current public contract.

Each tombstone should state:

- retired wording
- current owner surface
- whether any `human_doc:*` contract link still requires an in-place program brief
- where the reader should go for the current truth

Current truth:

- RedCube AI is a visual-deliverable domain agent first.
- OPL is the Codex-first, stage-led runtime framework and hosted integration path.
- Codex CLI is the default minimum concrete execution unit unless a hosted/proof backend is explicitly selected.
- Contract-linked old briefs stay in `docs/program/` with lifecycle notes until their machine-readable references move.
