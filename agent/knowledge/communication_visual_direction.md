# Communication And Visual Direction Knowledge

Communication strategy and visual direction are AI-first surfaces.

Required distinctions:
- Hard constraints are binding.
- Planning signals and examples are non-binding input context.
- Legacy prompt assets provide detailed task wording and schemas.
- Canonical stage policy lives in `agent/prompts/*.md`.
- Audience-facing deliverable text must not leak system fields, route names, operator notes, source ids, or internal labels.

Stage outputs are refs and accepted artifacts, not generic runtime success flags.
