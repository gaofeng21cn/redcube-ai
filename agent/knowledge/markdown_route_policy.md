# Markdown Route Policy

Owner: RedCube AI
Purpose: explicit optional refs-only policy for Markdown and Marp route material.

Policy:
- Markdown and Marp are optional route references for explicitly requested text-first, script, outline, or Marp deck material.
- Markdown and Marp routes are not the default RCA PPT deck route and do not replace `author_image_pages -> screenshot_review -> export_pptx`.
- Markdown and Marp route material is refs-only handoff material unless an RCA owner receipt authorizes artifact mutation in the active deliverable loop.
- External Markdown or Marp tooling may be referenced as material or packaging context only; it does not become RCA runtime authority or visual review/export authority.

Forbidden substitutions:
- Markdown or Marp completion cannot issue source-ready, visual-ready, exportable, handoffable, review-pass, or publication-ready verdicts.
- Markdown or Marp route refs cannot bypass screenshot review, export gates, repair refs, or owner receipt boundaries.
