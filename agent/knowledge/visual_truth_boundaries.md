# Visual Truth Boundaries

RCA is the owner for visual-domain truth. OPL generated surfaces consume descriptors and refs for discovery, scheduling, projection, and handoff.

RCA-owned bodies and verdicts:
- source readiness judgment
- communication strategy acceptance
- visual direction acceptance
- review/export verdict
- canonical artifact authority
- visual memory body and accept/reject verdict
- owner receipt

OPL-owned shells:
- generated CLI/MCP/Skill/product-entry/status/session/sidecar/workbench descriptors
- generic provider-backed attempt runtime
- typed queue, retry, dead-letter, and operator projection
- generic artifact gallery or handoff shell over RCA refs
