# Artifact And Export Authority

Artifact creation and export are RCA authority surfaces under route contracts.

Rules:
- Image-first PPT pages are complete 16:9 slide images unless an explicit HTML or native PPTX route is selected.
- Image-first Xiaohongshu pages are complete 3:4 note images unless an explicit HTML route is selected.
- Export packaging requires review gate refs, real conversion/package invocation refs, page or file count checks, and artifact locator refs.
- Native helper output is helper evidence until RCA records owner receipt refs or typed blockers.
- OPL generated handoff surfaces consume artifact refs and cannot write artifact bodies.
