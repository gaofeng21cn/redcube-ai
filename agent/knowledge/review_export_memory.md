# Review Export Memory Knowledge

Review, export, and memory decisions are RCA-owned.

AI-first review must inspect rendered evidence:
- final screenshots
- exported pages
- visual director review evidence
- source fidelity and visible copy boundaries
- route-specific packaging evidence

Visual memory writeback requires:
- a rendered artifact evidence ref
- review verdict ref
- proposal ref
- accept/reject owner receipt ref

OPL can project readiness gaps and memory receipt coverage, but it cannot accept or reject visual memory.
