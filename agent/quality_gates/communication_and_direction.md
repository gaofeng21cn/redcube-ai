# Communication And Direction Gate

Owner: RedCube AI
Purpose: protect communication strategy and visual direction decisions.

Gate rules:
- Storyline, outline, page plan, density, visual rhythm, and direction acceptance require AI-first stage artifacts.
- Planning signals, examples, schemas, and prior defaults are input signals, not approval contracts.
- Visual pattern memory, tool affordances, examples, and route precedents are advisory prompt context. They can shape attention and candidate comparison, but they cannot score routes, generate a winning layout by themselves, or approve direction.
- Audience-facing text must be free of route names, prompt fields, source ids, and operator-only wording.
- OPL may index accepted refs but cannot generate or override acceptance verdicts.
