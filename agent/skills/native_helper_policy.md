# Native Helper Skill Policy

Owner: RedCube AI
Purpose: constrain native PPT, Office, image, PDF, and package helpers to RCA authority boundaries.

Policy:
- Native helpers may materialize files only through RCA route policy, review/export gates, helper catalog refs, and owner receipt refs.
- Native helpers may report conversion failures, geometry evidence, package manifests, and artifact refs.
- Native helpers cannot generate visual ready, exportable, handoffable, communication acceptance, visual direction acceptance, or review verdicts.
- Helper output must remain in workspace/runtime artifact roots, not repo source directories.
